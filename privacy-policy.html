Privacy Policy

Last updated: April 30, 2026

HR Recruitment Search Keywords Helper is a Chrome extension that helps HR recruiters generate structured recruitment search keywords and target company suggestions from selected job titles on webpages.

Information the Extension Processes

The extension may process the job title text that the user selects on a webpage. This selected text is used only to generate recruitment search keywords and target company suggestions.

The extension stores user settings locally in the browser, including company information, target region, keyword language, API base URL, model name, and API key. These settings are used only to provide the extension's core functionality.

How Information Is Used

The selected job title and saved settings are used to generate recruitment search keywords and target company suggestions requested by the user. The extension does not use this information for advertising, tracking, profiling, or analytics.

Information Sharing

If the user configures an AI API endpoint, the selected job title and relevant settings may be sent to that user-configured API service to generate results. The extension does not sell user data and does not share user data with advertising services.

The developer does not operate a separate server to collect or store extension user data.

Local Storage

Extension settings are stored locally in the user's browser using Chrome storage. Users can remove this data by clearing the extension's stored data or uninstalling the extension.

Data Security

Users should configure only trusted API providers. When an HTTPS API endpoint is used, data is transmitted over a secure connection.

Limited Use

The extension uses user data only to provide or improve its single purpose: generating recruitment search keywords and target company suggestions from selected job titles. User data is not used or transferred for personalized advertising, retargeting, interest-based advertising, or unrelated purposes.
